MCG_Lite Hal Driver

Overview

The multi-purpose clock generator lite (MCG_Lite) module provides several clock source choices for the MCU. The MCG_Lite HAL provides a set of APIs to access these registers.

Get current clock output

The MCG_Lite HAL provides a set of APIs, which are dedicated to getting the different clock frequency. For example, MCGOUTCLK, MCGPCLK, MCGIRCLK and LIRC_CLK. Please make sure the mode setting registers have been configured properly, otherwise these APIs can not get valid value.
This is an example to get a default system reference clock:

#include "fsl_mcglite_hal.h"
// return frequncy value for specified clock name
uint32_t frequency = 0;
// get the current system default reference clock frequncy
frequency = CLOCK_HAL_GetOutClk();

Clock mode switching

MCG_Lite HAL provides APIs to switch between different clock modes. Before switching, please make sure the operation is valid, for example, switch between LIRC8M and LIRC2M directly is not allowed, so before switching to LIRC8M, please make sure current mode is not LIRC2M.

static mcglite_mcgoutclk_source_t CLOCK_HAL_GetClkSrcStat ( uint32_t  baseAddr)
inlinestatic

This function gets the Clock Mode Status. These bits indicate the current clock mode. The CLKST bits do not update immediately after a write to the CLKS bits due to internal synchronization between clock domains.

Parameters
baseAddrMCG_Lite register base address.
Returns
status Clock Mode Status
  • 00: HIRC clock is select.
  • 01: LIRC(low Internal reference clock) is selected.
  • 10: External reference clock is selected.
  • 11: Reserved.
static mcglite_ext_osc_status_t CLOCK_HAL_GetOscInit0 ( uint32_t  baseAddr)
inlinestatic

This function gets the OSC Initialization Status OSCINIT0. This bit, which resets to 0, is set to 1 after the initialization cycles of the crystal oscillator clock have completed. After being set, the bit is cleared to 0 if the OSC is subsequently disabled. See the OSC module's detailed description for more information.

Parameters
baseAddrMCG_Lite register base address.
Returns
OSC initialization status
